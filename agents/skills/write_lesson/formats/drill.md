# Format: Drill

**When to use:** The goal is repetition, calibration, or speed on an existing skill. Use when the user asks to "practice," "drill," "compare," or wants exercises that build fluency. A drill is small on purpose — it assumes the concept has been taught elsewhere.

## Structure Spine

### 1. Opening line

What skill this drills and why repetition matters here. Keep it to 1-2 sentences — the learner already knows the concept, they need to practice it.

### 2. The exercises

2-5 small items: rewrites, comparisons, classifications, predictions, or evaluations. Each exercise should be clearly bounded — the learner should know exactly what to produce.

Examples of good drill exercises:
- Rewrite a vague prompt into a testable brief
- Compare weak vs strong task framing
- Classify tasks by which model tier they need
- Predict what an agent will do given a prompt, then check

### 3. Answer key or evaluation criteria

How the learner knows they got it right. Can be exact answers, rubric criteria, or a "check your work by asking the agent to..." step.

### 4. Calibration note

What getting it wrong reveals about the learner's mental model. This is the teaching moment — not "you failed" but "if you chose X, it probably means you're thinking about Y wrong."

## Scope

2-5 exercises targeting one skill. Not a lesson — assumes the concept has been taught elsewhere. If the concept hasn't been taught, write a concept lesson first.

## Teaching Order

Exercise -> evaluate -> calibrate.

Do the thing, check if it's right, understand what mistakes reveal.

## Pre-Publish Checklist

| # | Check |
|---|-------|
| 1 | Are exercises small and clearly bounded? |
| 2 | Is there an answer key or clear evaluation criteria? |
| 3 | Does the drill target one specific skill? |
| 4 | Would getting it wrong reveal a real gap in understanding? |
| 5 | Is the formatting easy to scan on Discord? |

## Anti-Patterns

| Don't | Why |
|-------|-----|
| Write a drill that teaches a new concept | Drills practice existing knowledge — if the concept hasn't been taught, write a concept lesson first |
| Include more than 5 exercises | Too many exercises dilutes focus and fatigues the learner |
| Skip the calibration note | The calibration is what makes a drill a learning tool instead of just a quiz |
