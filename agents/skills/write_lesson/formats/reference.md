# Format: Reference

**When to use:** Lookup material — comparison tables, checklists, glossaries, prompt patterns, tool maps. Use when the learner needs to find information quickly, not learn a concept. Optimize for lookup, not reading.

## Template

### 1. What this reference covers

One sentence stating the domain this reference covers.

### 2. The reference material

Tables, checklists, glossaries, pattern libraries. Use the format that makes lookup fastest — usually tables or structured lists. Prose is a last resort.

### 3. When to use this (optional)

One line on when to look this up. Omit if obvious from the title.

## Scope

One lookup domain. If the reference covers multiple unrelated domains, split it.

## Pre-Publish Checklist

| # | Check |
|---|-------|
| 1 | Is the material optimized for lookup (tables, bullets, not prose)? |
| 2 | Is the scope clear from the first line? |
| 3 | Is the formatting easy to scan on Discord? |
